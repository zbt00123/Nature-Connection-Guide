> 数据来源：Open‑Meteo 全球气象服务（实时 / 历史 / 预报）

## ✨ 主要功能

- **查看任意日期 + 任意地点的地面材质适宜时段**
- **实时天气与历史 / 预报数据自动获取**
- **可视化年度圆盘**：滑动或点击选择日期
- **5 种常见地面材质**：草地、土地、沙滩、地砖、马路
- **智能数据标注**：估算 / 历史 / 预报 一目了然

---

## 🖱️ 如何使用

### 1. 选择日期

- **直接点击圆盘中心**：自动跳转到「今天」
- **拖动圆盘指针**：快速浏览全年不同日期的地面温度情况
- **使用日期时间选择器**：可精确到小时，用于查看特定时刻的实时温度

### 2. 选择地点

- **📍 定位按钮**：自动获取你当前所在城市
- **📌 城市按钮**：打开城市搜索面板  
  - 支持中文 / 拼音搜索（例如 “北京” / “Shanghai”）  
  - 也可直接点击下方热门城市快速切换

### 3. 读懂界面信息

#### 卡片
- **当前温度**：显示所选日期 + 小时的真实空气温度（或估算值）
- **自然足感卡片**  
  - 标题旁有 **数据标注**（估算 / 历史 / 预报）  
  - 显示空气温度、体感温度、地面温度的日范围  
  - 每种材质下方有一条 **时间条**：  
    - 🟢 绿色区间 = 温度适宜赤足（20～40℃）  
    - 🔵 蓝色 / 🟠 橙色 = 过冷或过热（不建议赤足）

#### 圆盘
- 代表一整年，不同颜色对应不同季节与温度趋势
- 指针指向当前选择的日期
- 外圈标注月份、节气，内圈显示选中日期

## ⚠️ 重要提示

- **赤足前请务必确认地面安全**（无尖锐物、无高温或低温伤害风险）
- 本工具提供的所有信息 **仅供参考**，不构成专业建议
- 初次尝试赤足请循序渐进，如感疼痛或不适请立即停止
