# 🌿 亲近自然指南 · 净足时光

> 通过实时气象数据和物理模型，帮助自然爱好者评估不同地面材质在全年中的赤足适宜时段。

## 🚀 在线预览

- 预览地址：[https://zbt00123.github.io/Nature-Connection-Guide/](https://zbt00123.github.io/Nature-Connection-Guide/)

## 📖 项目简介

「亲近自然指南」是一个交互式网页工具，用于指导用户在不同季节、不同天气条件下安全地进行赤足行走。它结合了：

- **实时气象数据**（空气温度、体感温度、地表温度）
- **地面材质热物理模型**（草地、土地、沙滩、地砖、马路）
- **年度季节圆盘** + 24小时适宜时段可视化

用户可以选择任意城市、任意日期（支持历史/未来7天预报），查看该日各材质的温度变化曲线，并获知推荐的赤足时间段。

## ✨ 主要功能

| 功能模块 | 描述 |
| --- | --- |
| 🌍 全球城市搜索 | 支持中/英文城市名称搜索，热门城市快捷选择 |
| 📍 自动定位 | 通过IP或浏览器定位，自动获取当地天气 |
| 🗓️ 日期/时间选择 | 可查看任意日期、任意小时的地面温度 |
| 🎡 年度季节圆盘 | 交互式圆盘，拖动指针或点击圆心快速跳转至今日 |
| 🌡️ 材质温度模型 | 基于太阳辐射、云量、热惯性估算5种常见地面材质温度 |
| 📊 适宜时段条 | 24小时时间线上高亮显示“适宜赤足”（28~35℃）的时段 |
| ⚠️ 安全风险提示 | 首次进入强制阅读赤足安全须知，特殊人群警告 |

## 🧠 技术栈 & API 依赖

- **前端**：原生 HTML5 / CSS3 / JavaScript (ES6)
- **可视化**：Canvas API（圆盘绘制 + 时间线绘制）
- **气象数据**：[Open-Meteo](https://open-meteo.com/)（实时预报、历史存档、地理编码）
- **反向地理编码**：[BigDataCloud](https://www.bigdatacloud.com/)（经纬度转城市名称）
- **辅助图标**：豆包AI 生成
- **交互算法**：DeepSeek / 通义千问

## 🖱️ 使用指南

1. **首次访问**：弹出“赤足安全须知”弹窗，阅读后点击“我已知晓”方可继续。
2. **选择城市**：点击左上角“📌 城市”按钮，搜索或从热门列表中选取。
3. **查看今日推荐**：圆盘指针默认指向当日，右侧面板显示5种材质的24小时适宜时段条。
4. **选择任意日期**：拖动圆盘指针（或直接修改日期时间选择器），即可查看该日各材质温度变化曲线。
5. **查看特定小时**：修改时间选择器中的小时（例如 14:00），左侧“当前温度”卡片会显示该小时各材质的瞬时温度及状态标签。
6. **定位功能**：点击“📍 定位”按钮，允许浏览器位置权限或自动IP定位。

## ⚠️ 重要安全提示

> 本工具所有地面温度均为基于气象数据的物理模型估算值，**不能替代实际测温**。赤足前请务必：
- 确认地面无尖锐物、化学污染、寄生虫风险
- 仅在温度适宜（28℃ ~ 35℃）且自身无糖尿病、神经病变等情况下尝试
- 初次赤足建议从短时间、柔软地面（草地）开始
- 如感到疼痛或过热/过冷，立即停止

详细风险说明请参见工具内置的“赤足安全须知”弹窗。
