# 亲近自然指南 · 净足时光

🌿 项目地址：[https://zbt00123.github.io/Nature-Connection-Guide/](https://zbt00123.github.io/Nature-Connection-Guide/)

「亲近自然指南」是一款帮助你在不同季节、不同地面材质上安全享受赤足体验的网页工具。它利用实时气象数据与物理模型，估算草地、土地、沙滩、地砖和马路等地面的温度，并结合日历与时间选择，给出每个时段的适宜等级与推荐时段。

---

## ✨ 主要功能

- **日历视图**  
  支持阳历/农历显示，显示节假日（中国法定假日及调休，自动标注“休/班”）、二十四节气，点击任意日期即可查看当日地面温度详情。

- **地面温度时间轴**  
  针对 5 种常见地面材质（🌱草地、🟫土地、🏖️沙滩、🧱地砖、🛣️马路）绘制 24 小时温度变化曲线，并高亮适宜赤足的温度区间（28℃～35℃）。鼠标悬停或移动端触摸可查看具体时刻温度。

- **当前时刻实时展示**  
  显示选定日期与时刻的空气温度、天气状况，以及各材质的估算表面温度与舒适度标签（过冷/适宜/较热/危险等）。

- **城市搜索与自动定位**  
  支持手动搜索全球城市（基于 Open-Meteo 地理编码），或通过浏览器 GPS / IP 定位自动获取当前位置。

- **深色模式**  
  点击右下角 🌙/☀️ 按钮切换深色/浅色主题，并可记住偏好设置。

- **安全风险提示**  
  首次访问弹出安全须知，强调赤足风险，用户确认后方可使用。页脚亦包含完整的安全提示与免责声明。

---

## 🚀 如何使用

1. 打开 [网页](https://zbt00123.github.io/Nature-Connection-Guide/)。
2. 阅读并确认安全风险弹窗。
3. 允许浏览器定位，或手动点击 **📌 城市** 按钮选择城市。
4. 在左侧日历面板中切换日期，或在顶部粘性工具栏中快速选择时间，“自然足感”面板会同步更新该时间下各地面温度的曲线与推荐时段。
5. 查看 **当前温度** 卡片，了解此刻各材质地面温度与体感等级。
6. 页脚提供了详细的数据来源、安全须知及免责声明，使用前请务必阅读。

## 🔧 技术说明

本项目使用纯前端技术构建，依赖以下优秀的开源服务与数据接口：

| 功能 | 项目/API | 链接 |
|------|----------|------|

## ⚠️ 安全须知与免责

赤足户外存在割伤、烫伤、冻伤、过敏及感染等风险。使用本工具前请确认：

- 🌡️ 地面温度适宜范围为 **28℃ ~ 35℃**，高温地面（＞45℃）可瞬间灼伤。
- 🧱 检查地面是否有玻璃、金属、尖锐物及化学污染。
- ☀️ 暴晒后地砖/沥青温度远超气温，注意防晒并缩短暴露时间。
- 🩺 糖尿病、神经病变、末梢循环障碍者请勿赤足。
- 🚸 儿童需在成人监护下进行，老人及孕妇谨慎尝试。
- 🌧️ 雨后地面湿滑，不建议赤足。
- ⚠️ 遇疼痛或不适立即停止。本指南仅供参考，使用者自行承担全部风险。
