# 🌿 亲近自然指南 · 净足时光

> 一款帮助你判断不同地面材质在全年中赤足适宜时段的交互式网页工具。  
> 结合实时天气数据与气候估算，直观展示草地、土地、沙滩、地砖、马路等材质在一天中适合光脚行走的时间段。

---

## 📌 核心功能

### 1. 全年时间圆盘
- **指针拖动**：用手指或鼠标拖动红色指针，即可选择任意日期，圆盘中央会显示当前选中的月日。
- **点击圆心**：电脑端，点击圆盘中央区域，快速跳转到“今天”的日期。

### 2. 实时天气数据
- **数据来源**：Open-Meteo 全球气象服务。
- **显示内容**：
  - 🌡️ 空气温度（当日最低～最高）
  - 🏃‍♂️ 体感温度（综合空气与地面温度估算）
  - 🔥 地面温度（土壤表层 0cm 温度）
- **数据标识**：明确标注“真实数据”或“估算数据”，让您了解数据来源。

### 3. 地面材质适宜时段
- **五种常见地面**：
  - 🌱 草地
  - 🟫 土地
  - 🏖️ 沙滩
  - 🧱 地砖
  - 🛣️ 马路
- **展示内容**：
  - 每种材质当天的温度范围（℃）
  - 一天中适合赤足的时间段（基于 20℃～40℃ 舒适区间）
- **颜色提示**：若某材质全天不适宜，文字会以橙色高亮显示。

### 4. 综合适宜时段时间轴
- **可视化时间轴**：以 24 小时为横轴，通过颜色渐变表示全天温度变化。
- **高亮区间**：叠加半透明白色块，标注五种材质综合叠加后适宜光脚的时段（取并集）。
- **时间标记**：每个适宜时段起止点会标注具体时间（如 `8:00-18:00`）。

### 5. 城市选择与自动定位
- **城市搜索**：
  - 支持中英文输入（如“北京”“Shanghai”）
  - 提供北京、上海、广州、深圳、杭州、成都等热门城市快捷按钮
- **自动定位**：
  - 首次打开网页时自动尝试获取您的位置（5 秒超时）
  - 若定位失败，居中弹出提示框，引导您手动选择城市
  - 点击“📍 定位”按钮可随时重新定位
- **定位服务**：采用 ip-api.com、ipapi.co 等多源 IP 定位 + 浏览器 Geolocation，并反向解析中文城市名。

### 6. 日期快捷切换
- **日期选择器**：支持手动选择任意日期（年月日）
- **“今天”按钮**：一键返回当前日期并刷新天气数据

---

## 🧭 使用说明

### 第一步：打开网页
直接在浏览器中访问 HTML 文件，或部署到 GitHub Pages、任何静态服务器。

### 第二步：选择城市
- **自动定位**：首次加载时会自动尝试定位（约 5 秒），若成功则自动切换为您所在城市。
- **手动选择**：
  - 点击“📌 城市”按钮，搜索城市名称或从热门城市中选择。
  - 确认后，页面会重新获取该城市当天的天气数据并刷新所有内容。

### 第三步：选择日期
- **拖动圆盘指针**：顺时针/逆时针拖动红色指针，选择任意日期。
- **点击圆心**：快速回到“今天”。
- **日历控件**：点击顶部的日期输入框，直接选择年/月/日。

### 第四步：查看适宜时段
- **左侧面板**：查看当前日期五种材质的温度范围及适宜时间段。
- **时间轴**：查看综合适宜时段的可视化分布，帮助规划出门时间。

## 🌐 数据来源与隐私说明

| 用途         | API 服务商              | 说明                           |
| ------------ | ----------------------- | ------------------------------ |
| IP 定位      | ip-api.com / ipapi.co   | 通过 IP 地址获取大致城市       |
| 经纬度转中文 | bigdatacloud.net        | 反向地理编码，返回中文城市名   |
| 天气数据     | open-meteo.com          | 空气温度、体感温度、地表温度   |
| 城市搜索     | open-meteo.com Geocoding | 根据关键词搜索城市坐标         |

- 所有 API 均为**免费服务**，无需注册或密钥。
- 不收集任何个人信息，定位仅用于获取当地天气。
- 若自动定位失败，页面会提示手动选择，不影响核心功能使用。

## ⚠️ 注意事项

1. **赤足安全**：本工具仅提供温度参考，实际赤足前请务必确认地面**无尖锐物、无高温烫伤风险、路面平整**。
2. **数据性质**：
   - 天气数据优先使用 Open-Meteo 实时预报（未来 7 天）。
   - 若请求超时或超出预报范围，自动切换为气候估算模型（基于纬度和历史平均气温）。
3. **网络要求**：需联网获取天气和定位数据，离线时仅展示估算数据。
4. **兼容性**：现代浏览器（Chrome、Safari、Edge、Firefox）均可正常使用，移动端支持触控拖动。
