# 亲近自然指南 · 净足时光

> 通过实时气象数据和物理模型，帮助自然爱好者评估不同地面材质在全年中的赤足适宜时段。

**🌐 网站地址：[https://zbt00123.github.io/Nature-Connection-Guide/](https://zbt00123.github.io/Nature-Connection-Guide/)**

## 项目简介

「亲近自然指南」是一个交互式网页工具，用于指导用户在不同季节、不同天气条件下安全地进行赤足行走。它结合了：

*   **实时气象数据**（空气温度、体感温度、地表温度）
*   **地面材质热物理模型**（草地、土地、沙滩、地砖、马路）
*   **24小时适宜时段可视化**

用户可以选择任意城市、任意日期（支持历史/未来7天预报），查看该日各材质的温度变化曲线，并获知推荐的赤足时间段。

## ✨ 主要功能

| 功能模块 | 描述 |
| --- | --- |
| 全球城市搜索 | 支持中/英文城市名称搜索，热门城市快捷选择 |
| 自动定位 | 通过IP或浏览器定位，自动获取当地天气 |
| 农历日历视图 | 交互式农历日历，支持年月快速跳转，显示节假日与节气信息 |

## 技术栈 & API 依赖

*   **前端**：原生 [HTML5](https://developer.mozilla.org/zh-CN/docs/Web/HTML) / [CSS3](https://developer.mozilla.org/zh-CN/docs/Web/CSS) / [JavaScript (ES6)](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript)
*   **可视化**：[Canvas API](https://developer.mozilla.org/zh-CN/docs/Web/API/Canvas_API)（时间线绘制）
*   **日历组件**：自研农历日历组件，集成 [lunar-javascript](https://github.com/6tail/lunar-javascript) 库
*   **气象数据**：[Open-Meteo](https://open-meteo.com/)（实时预报、历史存档、地理编码）
*   **反向地理编码**：[BigDataCloud](https://www.bigdatacloud.com/)（经纬度转城市名称）
*   **辅助图标**：[豆包AI](https://www.doubao.com/) 生成
*   **交互算法**：[DeepSeek](https://www.deepseek.com/) / [通义千问](https://tongyi.aliyun.com/)
*   **深色模式适配**：[Dark Reader](https://github.com/darkreader/darkreader)（开源项目鸣谢）

## 使用指南

1.  **首次访问**：弹出“赤足安全须知”弹窗，阅读后点击“我已知晓”方可继续。
2.  **选择城市**：点击左上角“🏙️ 城市”按钮，搜索或从热门列表中选取。
3.  **自动定位**：点击“📍 定位”按钮，允许浏览器位置权限或自动IP定位。
4.  **查看今日推荐**：中央日历卡片默认选中今日，右侧面板显示5种材质的24小时适宜时段条。
5.  **选择任意日期**：在日历卡片中点击任意日期，即可查看该日各材质温度变化曲线与适宜时段。
6.  **查看特定小时**：修改时间选择器中的小时（例如14:00），卡片会显示该小时各材质的瞬时温度及状态标签。
7.  **快捷跳转**：使用右上角悬浮按钮，可在日历、实时温度、材质详情面板间快速定位。
8.  **切换主题**：点击右上角🌙/☀️按钮，在深色与浅色模式间自由切换。

## ⚠️ 重要安全提示

> 本工具所有地面温度均为基于气象数据的物理模型估算值，**不能替代实际测温**。赤足前请务必：

*   确认地面无尖锐物、化学污染、寄生虫风险
*   仅在温度适宜（地面温度28℃ ~ 35℃）且自身无糖尿病、神经病变等情况下尝试
*   初次赤足建议从短时间、柔软地面（草地）开始
*   如感到疼痛或过热/过冷，立即停止

详细风险说明请参见工具内置的“赤足安全须知”弹窗。
