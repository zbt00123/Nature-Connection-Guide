# 亲近自然指南 · 净足时光

> 一款帮你安全、舒适地享受户外赤足体验的智能网页工具  
> **在线体验**：[https://zbt00123.github.io/Nature-Connection-Guide/](https://zbt00123.github.io/Nature-Connection-Guide/)

## 📖 项目简介

「亲近自然指南」利用**实时气象数据**与**地面热力学模型**，估算草地、土地、沙滩、地砖、马路等常见地面的全天温度变化。结合农历、节气、法定节假日及动态调休信息，以日历形式直观展示任意日期的赤足适宜时段，并给出 **28℃～35℃** 的安全推荐区间。

无论是规划春日踏青、夏日沙滩漫步，还是秋日草坪瑜伽，这款工具都能为你提供科学的赤足决策参考。

---

## ✨ 核心功能

### 📅 智能日历面板
- 支持**阳历/农历双显**，标注二十四节气、传统节日（春节、清明等）及公历节日。
- 接入中国法定节假日数据，自动标记 **“休”**（假日）/ **“班”**（调休上班日）。
- 点击任意日期，立即切换当日地面温度详情。
- 提供**年份/月份快速选择器**（点击年月文字展开），方便跨月、跨年浏览。
- 顶部 **粘性快捷栏** 在滚动时出现，可快速切换日期/时间并跳回“现在”。

### 🌡️ 24小时地面温度时间轴
- 针对 5 种常见材质（草地、土地、沙滩、地砖、马路），绘制**全天温度曲线**。
- **舒适时段**：温度落在 28℃～35℃ 区间时，时间轴对应区域会以绿色高亮显示。
- **鼠标悬停 / 触摸滑动**：实时显示某一时刻的精确温度，移动端同样流畅体验。
- 基于当地纬度、实时天气（晴/多云/雨雪等）及太阳辐射模型动态计算，比简单气温估算更贴近真实体感。

### 🌍 实时天气与当前时刻
- 展示选定时刻的**空气温度、体感温度、天气状况**（图标+文字）。
- 同步计算各地面材质的**估算表面温度**，并标注舒适度标签：❄️过冷 / ✅适宜 / 🔥过热 / ⚠️危险。
- 数据优先使用 **Open‑Meteo 实时预报（未来7天）或历史归档（过去90天）**，超出范围自动降级为气候学估算。

### 📍 城市搜索与自动定位
- **手动搜索**：支持全球城市中/英文名称搜索（基于 Open‑Meteo Geocoding）。
- **自动定位**：优先使用浏览器 GPS，失败后自动降级到 IP 定位（BigDataCloud 反向地理编码）。
- **热门城市管理**：
  - 预设多个国内热门城市。
  - 点击“编辑”进入编辑模式：**长按拖动**可调整顺序，**点击删除按钮**移除城市。
  - 支持**自由添加新城市**（搜索后自定义显示名称和图标）。

### 🎨 深色模式 & 导航
- 右上角 **🌙/☀️** 按钮一键切换深色/浅色主题。
- 滚动页面时，顶部**快捷栏**随时修改查询时间。
- 右侧悬浮导航按钮可快速定位到 **📅日历**、**🌡️当前温度**、**📍自然足感**、**⚠️安全须知**。

### ⚠️ 安全风险提示
- 首次访问时弹出**赤足安全须知**，用户确认后方可继续使用。
- 页脚包含详尽的风险提示与免责声明，涵盖温度区间、地面检查、特殊人群、雨后湿滑等关键注意事项。
- **任何信息均来自第三方**，本网页仅用于信息整合，如果网页信息有误，可去对应信息源进行提交更正。

## 🧠 技术原理简析

| 模块 | 说明 |
|------|------|
| **气象数据** | 优先使用 Open‑Meteo 的实时预报（未来7天）或历史归档（过去90天）；超出范围则采用基于纬度的气候学正弦曲线估算。 |
| **地面温度模型**   | 在气温基础上，综合考虑**太阳辐射强度**（根据纬度、日期、时刻、云量衰减）、**材质吸收率/热惯性**、**夜间辐射冷却**，采用一阶滞后滤波模拟热响应。 |
| **农历与节气** | 使用 `lunar-javascript` 库实现阳历 → 农历转换，并获取节气、传统节日名称。 |
| **节假日调休** | 接入 `holiday-cn` 开源数据集，动态标注当年法定假日及补班日。 |
| **深色模式** | 集成 Dark Reader，对页面整体亮度/对比度/棕褐色调进行平滑变换。 |
| **本地存储** | 保存深色模式偏好、热门城市列表、风险确认状态。 |

## 🚀 如何使用

1. 访问 [网页链接](https://zbt00123.github.io/Nature-Connection-Guide/)。
2. 阅读并同意安全风险提示。
3. 允许浏览器定位或手动选择城市。
4. 在日历中选择日期 → 查看地面温度时间轴中的**绿色高亮时段**（即为推荐赤足时间）。
5. 使用时间选择器微调时刻（支持半小时精度），查看对应温度与舒适度标签。
6. 将常用城市添加到“热门城市”列表，可编辑拖拽排序或删除。

> 💡 **小贴士**：移动端同样支持触摸滑动时间轴。

## ⚠️ 安全须知 & 免责声明

户外赤足存在割伤、烫伤、冻伤、过敏及感染等潜在风险。使用本工具前请务必知晓：

- 🌡️ 推荐赤足地面温度为 **28℃ ~ 35℃**，高于 45℃ 可造成瞬间灼伤，低于 15℃ 有冻伤风险。
- 🧱 实际活动前请**检查**地面，确保无玻璃、金属、尖锐物及化学污染。
- ☀️ 夏季暴晒后，深色地砖/沥青路面实际温度可能比气温高出 20℃ 以上，请勿直接赤足。
- 🩺 糖尿病、神经病变、末梢循环障碍者**严禁**赤足行走。
- 🚸 儿童须在成人监护下进行，老人及孕妇请谨慎尝试。
- 🌧️ 雨后地面湿滑，且可能隐藏尖锐物，不建议赤足。
- ⚠️ 一旦感到疼痛或不适，**立即停止**并检查足部。

> **本指南仅为决策参考，不构成医疗建议。使用者须自行承担全部风险。**
